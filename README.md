
## Instalação

* Criar arquivo .env de acordo com example.env
* Criar arquivo arquivo ormconfig.json de acordo com documentação https://docs.nestjs.com/techniques/database


```bash
$ npm install
```

```bash
$ npm run typeorm migration:run
```

## Iniciar aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
